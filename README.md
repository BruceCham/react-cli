### Instructions

Setup

```
// clone the repo
git clone https://github.com/brucecham/redux-saga.git

cd redux-saga

yarn
```

Run the demo

```
yarn start
```

build the demo

```
yarn build
```

Run tests

```
yarn test
```

update package.json

```
ncu -a
```

静态服务器

```
sts 8080
```
